Buttons

There are 3 ways to create buttons in xtyle. A button can be created using an anchor <a class="btn"> or <a class="button">, button <button>, and input type submit <input type="submit"> elements. All three ways will behave and look the same across all browsers. Add the class border-radius to create buttons with rounded corners.
