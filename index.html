Colors

Xtyle comes with an exclusive set of colors that can be used in your projects. Color classes are .yellow, .red, .blue, .green, .carbon, .silver, .white, and .black. You can also use the following hexadecimal notation codes directly on your HTML files.

Font

Xtyle uses the excellent Google's Roboto font as its default font. You can change this option at any time on top of the xtyle.css file.

Text-color

To change text color you just need to include one of the color classes in addition to the reserved class name .color. For example, if you want your paragraph with color green, you would write something like the following:

Lorem ipsum dolor sit amet, consectetur adipiscing elit. Sed tristique tellus et vulputate lacinia. Etiam sapien est, congue vitae ultricies non, eleifend sed justo. Ut feugiat diam est, sed fermentum elit mollis a. Aenean eu posuere dolor, in interdum elit.
<p class='color green'>...</p>
